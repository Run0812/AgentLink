## 初步规划

### 功能清单
1. **电影搜索** - 按标题搜索
2. **电影详情** - 导演、演员、剧情、评分
3. **分类浏览** - 类型、年份、评分排序
4. **海报展示** - 电影图片
5. **相似推荐** - 基于当前电影推荐

### 技术组件
- `SKILL.md` - 技能定义文件
- API 封装模块
- 格式化输出模块
- 配置管理 (API Key)

### 文件位置
- 技能文件: `.agents/skills/movie-search/SKILL.md`
- 可能的其他文件: `.agents/skills/movie-search/`
